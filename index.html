Jefer - Discord bot

Jefer is a discord bot for automated mute role at your discord guild. It's to add one command to your server. To add/edit the mute role just write "j!setMuteRole [role_name]" in any chat. If you want to change it later, only the server owner is able to do that is an other value is set.
